    Security Note: Your API keys are stored locally on your device using chrome.storage.local. They are only sent directly to the AI providers (Google, OpenAI, Anthropic) and never to a central server.

    How to get your Gemini API Key

    1. Visit Google AI Studio
    2. Sign in with your Google account
    3. Click "Get API Key" in the top menu
    4. Click "Create API Key" button
    5. Select or create a Google Cloud project
    6. Copy the generated API key
    7. Paste it in the field above

    How to get your OpenAI API Key

    1. Visit OpenAI API Keys
    2. Sign in to your OpenAI account
    3. Click "Create new secret key"
    4. Name the key and create it
    5. Copy the generated API key
    6. Paste it in the field above

    How to get your Anthropic Claude API Key

    1. Visit Anthropic API Keys
    2. Sign in to your Anthropic account
    3. Click "Create Key"
    4. Name the key and confirm creation
    5. Copy the generated API key
    6. Paste it in the field above
